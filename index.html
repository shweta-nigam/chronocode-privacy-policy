# Privacy Policy for ChronoCode **Last Updated:** June 23, 2026 ## Introduction ChronoCode ("we", "our", or "us") is a mobile application designed to help users save, organize, manage, import, export, and interact with code snippets using AI-powered assistance. This Privacy Policy explains how ChronoCode collects, uses, stores, and protects information when you use the application. By using ChronoCode, you agree to the practices described in this Privacy Policy. --- ## Information We Collect ### Information You Provide ChronoCode may process information that you voluntarily provide, including: * Code snippets you create, edit, or save * Prompts and questions submitted to AI features * Files selected for import into the application * Files selected for export or sharing ### Automatically Collected Information The application may collect limited technical information necessary for functionality and security, such as: * Device type * Operating system version * Application version * Crash and diagnostic information provided by the operating system or Google Play services --- ## How We Use Information We use information solely to: * Store and organize code snippets * Provide AI-powered coding assistance and explanations * Enable import, export, and sharing functionality * Improve application reliability and performance * Diagnose and fix technical issues --- ## AI Services ChronoCode provides AI-powered features through third-party AI providers. When you submit prompts, code snippets, or questions to AI features, that information may be transmitted to the AI service provider solely for the purpose of generating a response. Users should avoid submitting sensitive personal information, passwords, API keys, financial information, or confidential business data to AI features. --- ## Local Storage ChronoCode primarily stores user-created snippets and application data locally on the user's device using local database storage. Users maintain control over their locally stored content. --- ## File Import and Export ChronoCode allows users to: * Import code files and snippets from their device * Export snippets to files * Share exported files using device sharing features Files are processed only for the requested operation and are not uploaded to our servers. --- ## Data Sharing ChronoCode does not sell, rent, or trade personal information.
